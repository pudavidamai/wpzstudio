**Languages / 语言**:

이 저장소는 **WPZ Studio 공식 GitHub 저장소**로, 공식 웹사이트와 제품 정보를 유지 관리하기 위해 사용됩니다.  
우리는 생산성 도구, 유틸리티, 교육, 개발자 시나리오를 아우르는 **사용하기 쉽고, 신뢰할 수 있으며, 학습 비용이 낮은** 데스크톱 소프트웨어를 만드는 독립 개발 스튜디오입니다.

**공식 웹사이트**: <https://pudavidamai.github.io/wpzstudio/>  
**Microsoft Store(게시자)**: <https://apps.microsoft.com/search/publisher?name=WPZStudio>

## 우리가 만드는 것

WPZ Studio는 전 세계 사용자를 위한 데스크톱 앱을 지속적으로 출시하고 있으며, 핵심 방향은 다음과 같습니다.

1. 생산성(파일 관리, 터미널, 클립보드, 압축, PDF, Excel 비교 등)
2. 유틸리티 및 도구(네트워크 진단, 로그 분석, 시스템 정리 등)
3. 개발자 도구(예: 이벤트 로그 분석)
4. 교육 앱(수학, 파닉스, 타자, 시계 학습 등)
5. 엔터테인먼트 앱(예: 데스크톱 펫)
