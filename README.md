这是 **WPZ Studio 的官方 GitHub 仓库**，用于维护官网与产品信息。  
我们是一家独立开发工作室，专注于打造 **好用、可靠、学习成本低** 的桌面软件，覆盖效率工具、实用工具、教育学习与开发者场景。

**官网首页**：<https://pudavidamai.github.io/wpzstudio/>  
**Microsoft Store（发布者）**：<https://apps.microsoft.com/search/publisher?name=WPZStudio>

## 我们在做什么

WPZ Studio 持续发布面向全球用户的桌面应用，核心方向包括：

1. 高效工作（文件管理、终端、剪贴板、压缩、PDF、Excel 对比等）
2. 实用程序与工具（网络诊断、日志分析、系统清理等）
3. 开发人员工具（如事件日志分析）
4. 教育应用（数学、自然拼读、打字、时钟学习等）
5. 娱乐应用（如桌面宠物）
