## 私たちが作っているもの

WPZ Studio は、世界中のユーザー向けにデスクトップアプリを継続的に公開しており、主な方向性は次のとおりです。

1. 生産性向上（ファイル管理、ターミナル、クリップボード、圧縮、PDF、Excel 比較 など）
2. ユーティリティ・ツール（ネットワーク診断、ログ解析、システムクリーンアップ など）
3. 開発者ツール（例: イベントログ解析）
4. 教育アプリ（算数、フォニックス、タイピング、時計学習 など）
5. エンタメアプリ（例: デスクトップペット）
