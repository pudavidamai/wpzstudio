This is the **official GitHub repository of WPZ Studio**, used to maintain our website and product information.  
We are an independent development studio focused on building desktop software that is **easy to use, reliable, and quick to learn**, across productivity, utilities, education, and developer scenarios.

**Website**: <https://pudavidamai.github.io/wpzstudio/>  
**Microsoft Store (Publisher)**: <https://apps.microsoft.com/search/publisher?name=WPZStudio>

## What we build

WPZ Studio continuously publishes desktop applications for global users, with key directions including:

1. Productivity (file management, terminal, clipboard, compression, PDF, Excel comparison, etc.)
2. Utilities and tools (network diagnostics, log analysis, system cleanup, etc.)
3. Developer tools (such as event log analysis)
4. Education apps (math, phonics, typing, clock learning, etc.)
5. Entertainment apps (such as desktop pets)
