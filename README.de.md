## Was wir entwickeln

WPZ Studio veröffentlicht kontinuierlich Desktop-Apps für Nutzer weltweit. Die Schwerpunkte sind:

1. Produktivität (Dateiverwaltung, Terminal, Zwischenablage, Komprimierung, PDF, Excel-Vergleich usw.)
2. Utilities und Tools (Netzwerkdiagnose, Log-Analyse, Systembereinigung usw.)
3. Entwickler-Tools (z. B. Event-Log-Analyse)
4. Bildungs-Apps (Mathematik, Phonics, Tippen, Uhrzeitlernen usw.)
5. Unterhaltungs-Apps (z. B. Desktop-Haustiere)
