Ce dépôt est le **dépôt GitHub officiel de WPZ Studio**, utilisé pour maintenir le site web et les informations produit.  
Nous sommes un studio de développement indépendant, axé sur des logiciels de bureau **faciles à utiliser, fiables et rapides à prendre en main**, couvrant la productivité, les utilitaires, l'éducation et les scénarios développeur.

**Site officiel** : <https://pudavidamai.github.io/wpzstudio/>  
**Microsoft Store (éditeur)** : <https://apps.microsoft.com/search/publisher?name=WPZStudio>

## Ce que nous développons

WPZ Studio publie en continu des applications desktop pour des utilisateurs du monde entier, avec les axes principaux suivants :

1. Productivité (gestion de fichiers, terminal, presse-papiers, compression, PDF, comparaison Excel, etc.)
2. Utilitaires et outils (diagnostic réseau, analyse de logs, nettoyage système, etc.)
3. Outils développeur (comme l'analyse des journaux d'événements)
4. Applications éducatives (mathématiques, phonics, dactylographie, apprentissage de l'heure, etc.)
5. Applications de divertissement (comme les animaux de bureau)
